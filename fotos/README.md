# Para ti, mi amor

Sitio estático con carrusel y cubo 3D de fotos.  
Estructura:

- index.html — página principal
- fotos/ — carpeta con imágenes utilizadas

Abrir en el navegador: abrir `index.html` con un servidor estático o directamente (recomendado: usar `live-server` o `python -m http.server`).
